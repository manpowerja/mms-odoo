Inventory Tutorials

Full Receival on Purchase Order

  1. Open the inventory module.
  2. Go to the receipt operation at your location.
  3. Search for the purchase order number the recieval is to be done on.
  4. Select the receival operation that corresponds with the purchase order number (From Vendor to Input).
  5. Click the set quanities tab.
  6. Click the validate button.
  7. The pick operation has been completed.
  8. Open the inventory overview tab.
  9. Go to the internal transfer operation for your location.
  10. Search for the purchase order number the recieval is to be done on.
  11. Select the quality control operation that corresponds with the PO number (From Input to Quality Control).
  12. Click the set quanities tab.
  13. Click the validate button.
  14. The quality control operation has been completed.
  15. Open the inventory overview tab.
  16. Go to the internal transfer operation at your location.
  17. Search for the purchase order number the recieval is to be done on.
  18. Select the receival operation that corresponds with the purchase order number (From Quality Control to Stock).
  19. Click the set quanities tab.
  20. Click the validate button.
  21. Full receival from purchase order has been successfuly completed.

Partial Recieval on Purchase Order (With BackOrder)

  1. Open the inventory module.
  2. Go to the receipt operation that corresponds with your location.
  3. Search for the purchase order number the recieval is to be done on.
  4. Select the receival transaction that corresponds with the purchase order number (From vendor to input).
  5. Click the list button next to the products.
  6. Select the add a line tab.
  7. Enter the amount of the product that was received.
  8. Select the confirm button.
  9. Repeat step 5-8 for all products.
  10. Click the validate button.
  11. Click the create backorder button.
  12. The input operation has been completed.
  13. Click the inventory overview tab.
  14. Go to the internal transfer operation for your location.
  15. Search for the purchase order number the recieval is to be done on.
  16. Select the quality control operation that corresponds with the PO number (From Input to Quality Control).
  17. Select the set quanities tab.
  18. Click the validate button.
  19. Click the create backorder button.
  20. The quality control operation has been completed successfully.

  24. Select the receival operation that corresponds with the purchase order number (From Quality Control to Stock).
  25. Open the set quanities tab.
  26. Click the validate button.
  27. Click the create backorder button.
  28. The receival process has been completed successfully.

Full Delivery on Non-Contractual Sales Order

  1. Open the inventory module.
  2. Go to the pick operation at your warehouse.
  3. Search for the sales order number delivery is to be done on.
  4. Select the pick operation (From Stock to Packing Zone).
  5. Click the set quanities tab.
  6. Click the validate button.
  7. Pick operation has been completed.
  8. Go to the inventory overview tab.
  9. Go to the pack operation of your location.
  10. Search for the sales order number delivery is to be done on.
  11. Click the pack operation (From Packing Zone to Output).
  12. Click the set quanities tab.
  13. Click the validate button.
  14. Pack operation has been completed.
  15. Open the Inventory Overview tab.
  16. Go to the delivery orders operation that corresponds with your location.
  17. Search for the sales order number delivery is to be done on.
  18. Select the delivery operation (From Ouput to Customers)
  19. Select the set quanities tab.
  20. Click the validate button.
  21. Click the print button.
  22. Slect the delivery slip option and save document.
  23. Fulll delivery has been successfully completed.

Partial Delivery on Non-Contractual Sales Order (With Backorder)

  1. Open the inventory module.
  2. Go to the pick operation that corresponds with your location.
  3. Search for the sales order number delivery is to be done on.
  4. Select the pick operation (From Stock to Packing Zone)
  5. Click the done column for the products.
  6. Enter the amount that will be delivered on this delivery in the done column.
  7. Enter the done quantity for all products.
  8. Click the validate button.
  9. Click the create backorder button.
  10. Pick process has been completed.
  11. Click the inventory overview tab.
  12. Go to the pack operation that corresponds with your location.
  13. Search for the sales order number delivery is to be done on.
  14. Select the packing operation that corresponds with sales order number (From Packing Zone to Output).
  15. Click the set quanities tab.
  16. Click the validate button.
  17. Click the create backorder button.
  18. Packing operation has been completed.
  19. Click the inventory overview tab.
  20. Go to the delivery orders operation for your location.
  21. Search for the sales order number delivery is to be done on.
  22. Select the delivery operation that corresponds with requested sales order (From output to customer)
  23. Click the set quanities tab.
  24. Click the validate button.
  25. Click the create backorder button.
  26. Click the print button.
  27. Select the delivery slip option and save delivery slip.
  28. Partial devlivery with back order has been successfully completed.

Delivery on Contract Sales Orders (Billable and Non-Billable)

  1. Go to the sales module.
  2. Clear the My Quotation filter in the search bar.
  3. Search for the sales order number for the contract sale delivery is to be done on.
  4. Click the sales order that corresponds with the sales order number.
  5. Make note of the items on the sales order.
  6. Go to the dashboard.
  7. Go to the inventory module.
  8. Click the three dots on the internal transfer for your loaction.
  9. Select the planned transfer option (To be done based on sales order in step 5).
  10. Select the add a line link.
  11. Fill out all the relevant information for each product (product, demand, unit of measure).
  12. Add all the products needed to fulfil contract sales order.
  13. Add the contact.
  14. Change the destination location to the manpower contracts location for your warehouse.
  15. Click the save button.
  16. Click the mark as todo button.
  17. Click the set quantities tab.
  18. Click the validate button.
  19. Click the inventory overview tab.
  20. Go to the manpower contracts operation for your location.
  21. Clear the ready filter from search bar.
  22. Search for the sales order number delivery is to be done on.
  23. Select the delivery operation that corresponds with requested sales order.
  24. Click the check availability button.
  25. Click the set quantities tab.
  26. Click the validate button.
  27. Click the print button.
  28. Select the delivery slip option and save document.
  29. The delivery on the constract sales order has been successfully completed.

Internal Transfer to another location

  1. Go to the inventory module.
  2. Click the three dots on the internal transfer operation of location ITEMS ARE GOING TO.
  3. Select the planned transfer option.
  4. Select the add a line link.
  5. Fill all the relevant fields(product, demand, unit of measurement)
  6. Add all the products that is to be transferred to the other location.
  7. Select the contact and source location. The source location should be the stock location for your warehouse.
  8. Click the save button.
  9. Click the mark as todo button.
  10. Click the set quanities tabs.
  11. Select the validate button.
  12. Select the print button.
  13. Select the delivery slip option and save document.
  14. Internal transfer has been completed successfully.
