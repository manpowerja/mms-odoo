Administrator Tutorials

Add a user

  1. Open the settings module.
  2. Click the create button.
  3. Add user's name and email address.
  4. Specify their level of access for various modules.
  5. Click the save button.
  6. User has successfuuly been added. The system has sent an invitation to the user's email address.

Add new employee

  1. Open the employees module.
  2. Click the create button.
  3. Fill all relevant fields (Name, Position, Work Mobile, Work Email, Department, Manager etc.)
  4. Go to the hr settings tab.
  5. Fill out all fields(Employee Type, Related User). If there is no related employee, that field can be left null.
  6. Click the save button.
  7. New employee has been successfully added.

Add new employee to Point to Sales

  1. Open the Point of Sales module.
  2. Click on the three dots on the Point of Sales Location employee is to be added to.
  3. Click on the settings option.
  4. Enter the employees name in the authorized employees field.
  5. Click the save button.
  6. Employee has been successfully added to the Point of Sales.

Change pin for POS

  1. Ask user to click on their name on the dashboard.
  2. Select the my profile option.
  3. Select the hr setting tab.
  4. Click the edit button.
  5. Enternew pin in the pin field.
  6. Click the save button.
  7. User's POS pin has been updated.

Create new pricelist

  1. Open the sales module.
  2. Open the products tab.
  3. Select the pricelist option.
  4. Click the create button.
  5. Enter the name of the pricelist.
  6. Click the add a line tab to add each criteria for this pricelist. This can be applied to all products, a specific product or a group of products.
  7. Select the options relevant to the rule you want to apply.
  8. Click the save and close button.
  9. Once you have finished adding all rules, click the save button.
  10. New pricelist has been successfully created.

Edit pricelist

  1. Open the sales module.
  2. Open the products tab.
  3. Select the pricelist that needs to be editted.
  4. Click the add a line tab to add each criteria for this pricelist. This can be applied to all products, a specific product or a group of products.
  5. Select the options relevant to the rule you want to apply.
  6. Click the save and close button.
  7. After making all edits, click the save button
  8. Pricelist has been successfully editted.

Create new journal

  1. Open the accounting module.
  2. Open the configuration tab.
  3. Select the jornal option.
  4. Click the create button.
  5. Open the name and type of jornal.
  6. Fill all relevant information for the journal.
  7. Only for bank and cash journals: Go to incoming payments tab.
  8. Only for bank and cash journals: Click on the three dots.
  9. Only for bank and cash journals: Click the checkbox to make outstanding receipts accounts column.
  10. Only for bank and cash journals: Enter outstanding receipts account.
  11. Only for bank and cash journals: Go to outgoing payments tab.
  12. Only for bank and cash journals: Click on the three dots.
  13. Only for bank and cash journals: Click the checkbox to make outstanding payments accounts column.
  14. Only for bank and cash journals: Enter outstanding payment account.
  15. Click the save button.
  16. Jornal has been sucessfully created.

Create Report Template.

  1. Open any of the modules
  2. Click the checkbox beside records you want report formatter to capture.
  3. Click the action tab.
  4. Select the export option.
  5. Select the fields ypiu want to be visible in the report.
  6. Click the down arrow in the template field.
  7. Select the new template option.
  8. Type the name to be assigned to the template.
  9. Click the save button.
  10. Template has successfully been created.
