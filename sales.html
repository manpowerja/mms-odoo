Sales Tutorials

Create Non-Contractual sales order

  1. Open the sales module.
  2. Click create to create quotation.
  3. Fill in all relevant fields (customer, invoice and delivery address).
  4. Click add product to add each product.
  5. Fill out all relevant fields for each product.
  6. Click save to create quotation.
  7. Confirm quotation to create sales order.
  8. Sales Order has been sucessfully created.

Create Billable Sales Orders

  1. Open the sales module
  2. Click create to create quotation.
  3. Select Manpower and Maintainence Services Ltd. for the customer and invoice address.
  4. Select contract client as deliver address.
  5. Select Billable Contracts pricelist.
  6. Go to Other Info.
  7. Add Tag in the following format: Client Name/Branch/Billable
  8. Add customer reference (the purchase requistion number from the client)
  9. Go back to Order Lines and click on the three dots.
  10. Add the route column.
  11. Click add product for each product.
  12. Fill out all relevant fields (product, quantity, unit of measure, route). Route should be selected in accordance with your warehouse.
  13. Ensure the route is selected for all products.
  14. Click Save to create quotation.
  15. Click confirm to create sales order.
  16. Billable Sales Order has been successfully created.
